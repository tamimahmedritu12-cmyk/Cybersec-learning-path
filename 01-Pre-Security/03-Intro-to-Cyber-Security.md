##  Key Technical Concepts

### 1. Offensive vs. Defensive Security
* **Offensive Security (Red Teaming):** Focuses on breaking into systems, finding vulnerabilities, and exploiting them legally to test defenses (e.g., Ethical Hacking, Penetration Testing).
* **Defensive Security (Blue Teaming):** Focuses on protecting digital assets, monitoring network traffic, analyzing logs, and responding to security incidents (e.g., SOC Analyst, Incident Responder).

### 2. Google Dorking (Advanced Search Skills)
Using specific search operators in Google to find exposed sensitive data or vulnerable websites that are indexed publicly.

* **Useful Operators:**
  * `site:` - Limits results to a specific domain (e.g., `site:nasa.gov`).
  * `filetype:` - Searches for specific file extensions (e.g., `filetype:pdf "password"`).
  * `intitle:` - Finds pages with specific keywords in the title.

## Key Takeaway for Cybersecurity
Security requires a balance of both offensive mindset (knowing how attackers think) and defensive skills (knowing how to monitor and log malicious activities). Advanced OSINT and search skills save time during investigation phases.
